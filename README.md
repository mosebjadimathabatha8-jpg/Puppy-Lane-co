**Puppy Lane & Co** is a simple multi-page website for a local pet spa (grooming & pampering services) in Polokwane.

## Project Structure

- `Pages/` - HTML pages for the site
  - `index.html` (Home)
  - `about.html` (About)
  - `services.html` (Services)
  - `contact.html` (Contact)
  - `proposal.html` (Project Proposal)
- `Assets/` - Images used by the website
  - `Assets/Logo/logo.jpeg`
  - `Assets/Photos/*` (site photos)
- `style.css` - Main stylesheet for the site (referenced by all pages)
- `icons.css` - Intended icon stylesheet (referenced by HTML pages)

## Pages

- **Home (`index.html`)**: Welcome message + featured image.
- **About (`about.html`)**: Company overview, mission, vision, and target audience.
- **Services (`services.html`)**: List of services with prices and discounts.
- **Contact (`contact.html`)**: Email/phone/address details + a simple contact form.
- **Project Proposal (`proposal.html`)**: Goals/objectives, KPIs, design & UX, and technical requirements.

## How to Run

Because this is a static HTML/CSS project, you can run it by simply opening the pages in a browser.

### Recommended (Local)

1. Open the `Puppy Lane & Co/Pages/` folder.
2. Double-click `index.html` (or use your browser’s **File → Open**).
3. Navigate to other pages using the header menu.

> Note: All links and images use relative paths (e.g. `../Assets/...`), so keep the folder structure as-is.

## Notes

- The HTML files reference `icons.css` (`<link rel="stylesheet" href="icons.css">`). If you notice missing styling/errors in the browser, ensure `icons.css` exists at the expected location.
